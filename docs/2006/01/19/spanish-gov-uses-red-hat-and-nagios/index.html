Spanish Gov. uses Red Hat and NAGIOS

Jan 19, 2006 in UNCATEGORIZED
1 min read

Reading the spanish newspaper el mundo you can find information about the new application platform to link all the different administrations of the Spanish goverment Sevilla anuncia un nuevo ‘cerebro’ para la Administración (click on it to read it translated to english by google). Ok, news about a new system in the administration is not big deal…
But if you analize the picture of the TFT screen, you can see how they are using NAGIOS and Red Hat Linux. You can see some Sun servers too(this is not new since Sun is a big provider of Spanish administrations).
Everybody accepts Linux today as an Enteprise OS, but what about NAGIOS? Is it good enough to fight in the arena of products like HP Openview or BMC Patrol? It seems so.
